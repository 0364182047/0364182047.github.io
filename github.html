Hướng Dẫn Tạo Trang Web Cá Nhân Miễn Phí với GitHub Pages

Những bước cơ bản để đưa trang web của bạn lên mạng Internet.

1. GitHub Pages là gì?

GitHub Pages là một dịch vụ lưu trữ (hosting) trang web tĩnh miễn phí được cung cấp bởi GitHub. Nó lấy các tệp HTML, CSS, và JavaScript trực tiếp từ một kho chứa (repository) trên GitHub và tự động xuất bản chúng thành một trang web.

  • Miễn phí: Bạn không phải trả bất kỳ chi phí nào cho việc hosting.
  • Dễ dàng: Rất dễ để bắt đầu, đặc biệt nếu bạn đã quen với GitHub.
  • Phù hợp cho: Trang portfolio, blog cá nhân, trang giới thiệu dự án, tài liệu hướng dẫn...

2. Các bước thực hiện

Bước 1: Tạo tài khoản GitHub

Nếu chưa có, hãy đăng ký một tài khoản miễn phí tại github.com. Đây là nền tảng để bạn lưu trữ mã nguồn và trang web của mình.

Bước 2: Tạo một kho chứa (Repository) đặc biệt

Đây là bước quan trọng nhất. Tên của kho chứa phải tuân theo một định dạng rất cụ thể.

  1. Trên trang chủ GitHub, nhấn vào nút "New" để tạo repository mới.
  2. Tại ô "Repository name", bạn phải đặt tên theo đúng cú pháp sau:
    ten-dang-nhap.github.io

    (Thay ten-dang-nhap bằng tên đăng nhập GitHub của chính bạn. Ví dụ: nếu tên của bạn là bsphamhailinh, tên repo sẽ là bsphamhailinh.github.io).

  3. Đảm bảo repository được đặt ở chế độ "Public".
  4. Nhấn nút "Create repository".

Bước 3: Thêm tệp `index.html`

Mọi trang web đều cần một tệp "cửa ngõ" là index.html. Đây là trang sẽ hiển thị khi có người truy cập vào địa chỉ web của bạn.

  1. Trong repository bạn vừa tạo, nhấn vào "Add file" > "Create new file".
  2. Đặt tên tệp là index.html.
  3. Dán đoạn mã HTML cơ bản dưới đây vào phần nội dung:
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang của tôi</title>
</head>
<body>
    <h1>Chào mừng đến với trang web của tôi!</h1>
    <p>Đây là trang web được host trên GitHub Pages.</p>
</body>
</html>

Bước 4: Lưu thay đổi (Commit file)

Sau khi đã thêm mã, bạn cần lưu lại. Trong GitHub, hành động này gọi là "Commit".

  1. Kéo xuống cuối trang, bạn sẽ thấy phần "Commit new file".
  2. Bạn có thể để mô tả mặc định hoặc viết mô tả của riêng mình.
  3. Nhấn nút màu xanh "Commit new file".

Bước 5: Kích hoạt và truy cập trang web

Với repository có tên đặc biệt như trên, GitHub Pages thường được kích hoạt tự động.

Chờ một chút!

Quá trình xuất bản trang web có thể mất từ 1 đến 5 phút. Đừng lo lắng nếu nó chưa hoạt động ngay lập tức.

Sau vài phút, bạn có thể truy cập trang web của mình tại địa chỉ:

https://ten-dang-nhap.github.io

Nếu trang web không hiển thị, hãy kiểm tra cài đặt: Vào Settings > Pages trong repository của bạn. Tại mục "Source", đảm bảo bạn đã chọn branch là main (hoặc master) và nhấn Save.

3. Mẹo và thông tin thêm

  • Cập nhật trang web: Rất đơn giản! Bạn chỉ cần chỉnh sửa các tệp tin trong repository hoặc thêm tệp mới, sau đó "Commit" thay đổi. GitHub sẽ tự động cập nhật lại trang web cho bạn.
  • Sử dụng Git trên máy tính: Để làm việc chuyên nghiệp hơn, bạn nên cài đặt Git và học các lệnh cơ bản (clone, add, commit, push) để quản lý mã nguồn từ máy tính cá nhân.
  • Tên miền riêng: GitHub Pages cho phép bạn trỏ tên miền riêng (ví dụ: www.ten-mien-cua-ban.com) vào trang web của mình. Bạn có thể cấu hình trong phần Settings > Pages > Custom domain.

Chúc bạn thành công với trang web cá nhân của mình!
